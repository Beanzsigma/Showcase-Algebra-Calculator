# Showcase-Calculator        
VIDEO DEMO INCLUDED IN REPOSITORY
Using everything I've learn so far in Python to make a proper, functional algebra Calculator.

Description: The showcase algebra calculator is a Python based desktop application, built using CustomTkinter, helping it achieve the glitchy theme. There are four main tools in the calculator, which is the area mode, quadratic equation solver, factorial calculator, and the algebraic expression solver. The area mode supports many shapes, such as circles, rectangles, right triangles, any triangle, and regular polygons. The quadratic mode takes the coefficients A, B, and C, returning rounded solutions. The factorial mode allows you to calculate factorials up to 34, before displaying an overflow warning. The algebra mode allows users to input equations with the variable x, returning an approximate and rounded solution. The UI is designed with a dark and light colour scheme, with an animated glitch style GIF on each mode.
